I made a modified Linux Mint ISO with Cubic.

Feautures:
- Xanmod 6.6.16 v2 kernel added
- Some system apps removed
  
Added Apps:
- Steam
- Wine, Winetricks and Protontricks
- vkBasalt
- Lutris
- Feral Gamemode
- Goverlay and Mangohud
- Discord
- VLC
- qBitTorrent
- OnlyOffice
- Okular
- Pulse Effects
- Youtube Music Modded Client by th-ch
- Java Jre 8, 11 and 17

Removed Apps:
- Timeshift
- Transmission
- Celluloid
- LibreOffice
- Hexchat
- Hypnotix
- Onboard
- Redshift
- Sticky
- Thingy
- Thunderbird
- Warpinator
